This folder contains folder of the code for the different tools/datasets used for annotation.

corpus_code: With the use of google speech to text, the friends corpus (https://github.com/emorynlp/character-mining) was used and timestamps for the start and end of the utterances was added

data: different code used for the converting/processing data

exploration: different used over the course of time that was not kept. Inspiration and testing can be found there

features: not used

models: not used

pyscene_code: code useful for Pyscene, one bash script that can generate scene cuts annotation

visualization: not used
